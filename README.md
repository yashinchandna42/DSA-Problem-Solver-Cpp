# DSA Problem Solver (C++)

A menu-driven C++ application that demonstrates core Data Structures and Algorithms concepts such as sorting and searching.

---

## Features
- Bubble Sort implementation
- Linear Search implementation
- Displays time and space complexity
- Beginner-friendly and clean code

---

## Tech Stack
- Language: C++
- Concepts: Arrays, Sorting, Searching, STL

---

## How to Run
1. Open the project in VS Code
2. Compile using:
   g++ main.cpp
3. Run the program:
   ./a.exe
